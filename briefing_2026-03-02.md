## AMENAZAS Y CAMPAÑAS ACTIVAS

### ScarCruft Explota Zoho WorkDrive para C2 y Breaches de Redes Air-Gapped

**Nivel de Relevancia: CRÍTICA**

El APT norcoreano ScarCruft ha desplegado nuevas herramientas incluyendo un backdoor que utiliza Zoho WorkDrive como canal de comando y control (C2) para obtener payloads adicionales, y un implante que usa medios removibles para comunicarse y comprometer redes aisladas. La campaña "Ruby Jumper" marca una sofisticación operacional significativa en la evasión de defensas perimetrales tradicionales.

[ScarCruft Uses Zoho WorkDrive and USB Malware to Breach Air-Gapped Networks](https://thehackernews.com/2026/02/scarcruft-uses-zoho-workdrive-and-usb.html)

## VULNERABILIDADES DESTACADAS

### CVE-2026-20127: Cisco SD-WAN Explotado desde 2023

**Nivel de Relevancia: CRÍTICA**

Flaw de máxima severidad (CVSS 10.0) en Cisco Catalyst SD-WAN Controller y Manager permite bypass de autenticación a atacantes no autenticados para acceso administrativo. Explotación activa confirmada desde 2023, indicando distribución extendida sin parches en infraestructura operativa.

[Cisco SD-WAN Zero-Day CVE-2026-20127 Exploited Since 2023 for Admin Access](https://thehackernews.com/2026/02/cisco-sd-wan-zero-day-cve-2026-20127.html)

### CVE-2026-25108: FileZen OS Command Injection Bajo Explotación Activa

**Nivel de Relevancia: ALTA**

CISA confirmó explotación activa de vulnerabilidad de command injection en FileZen (CVSS 8.7) que permite ejecución de comandos del SO a usuarios autenticados. Inclusión en catálogo KEV oficial indica distribución activa en entornos productivos.

[CISA Confirms Active Exploitation of FileZen CVE-2026-25108 Vulnerability](https://thehackernews.com/2026/02/cisa-confirms-active-exploitation-of.html)

### SolarWinds Serv-U: 4 Flaws Críticos (CVE-2025-40538 y otros)

**Nivel de Relevancia: ALTA**

Cuatro vulnerabilidades críticas (CVSS 9.1) en Serv-U 15.5 permiten RCE, incluyendo broken access control que facilita creación de usuarios administrativos. Software de transferencia de archivos ampliamente desplegado en operaciones de supply chain.

### ClawJacked: Vulnerabilidad en OpenClaw AI Agent

**Nivel de Relevancia: ALTA**

Flaw crítica en el agente IA OpenClaw permitía a sitios maliciosos conectarse a instancias locales mediante WebSocket y tomar control. La vulnerabilidad reside en el sistema gateway central sin dependencias de plugins o extensiones de usuario.

[ClawJacked Flaw Lets Malicious Sites Hijack Local OpenClaw AI Agents via WebSocket](https://thehackernews.com/2026/02/clawjacked-flaw-lets-malicious-sites.html)

### Claude Code: Vulnerabilidades de RCE y Exfiltración de Credenciales

**Nivel de Relevancia: ALTA**

Múltiples flaws en el asistente de codificación Claude de Anthropic explotaban hooks, servidores MCP y variables de entorno para lograr ejecución remota de código y robo de credenciales API. Impacto potencial en cadenas de suministro de desarrollo de software.

[Claude Code Flaws Allow Remote Code Execution and API Key Exfiltration](https://thehackernews.com/2026/02/claude-code-flaws-allow-remote-code.html)

### RoguePilot: Fuga de GITHUB_TOKEN en GitHub Codespaces
