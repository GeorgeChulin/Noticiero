Esta actividad demuestra sofisticación en técnicas de evasión de aire-gapped y exfiltración de datos mediante servicios legítimos comprometidos.

## AMENAZAS RELACIONADAS CON IA

### ClawJacked: Falla de alta severidad permite hijacking de agentes OpenClaw locales via WebSocket

**ALTA**

Vulnerabilidad en OpenClaw permitía a sitios web maliciosos conectarse a instancia AI ejecutándose localmente y asumir control total. La falla residía en el sistema core (gateway OpenClaw) sin requerir plugins. Representa primer vector de compromiso confirmado de agentes AI locales a través de navegador.

[ClawJacked Flaw Lets Malicious Sites Hijack Local OpenClaw AI Agents via WebSocket](https://thehackernews.com/2026/02/clawjacked-flaw-lets-malicious-sites.html)

### RoguePilot: Vulnerabilidad AI en GitHub Codespaces filtraba GITHUB_TOKEN via Copilot

**ALTA**

Falla en GitHub Codespaces permitía inyectar instrucciones maliciosas en issues de GitHub para que Copilot revelaera GITHUB_TOKEN y permitiera control de repositorios. Representa primer caso confirmado de compromiso de repositorio via prompt injection en asistente AI. Parcheado por Microsoft.

[RoguePilot Flaw in GitHub Codespaces Enabled Copilot to Leak GITHUB_TOKEN](https://thehackernews.com/2026/02/roguepilot-flaw-in-github-codespaces.html)
