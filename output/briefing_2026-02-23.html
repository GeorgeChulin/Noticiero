1. VULNERABILIDADES CRÍTICAS (CVEs)

ClawJacked: Fallo de Seguridad Alta en OpenClaw AI

Relevancia: Alta

OpenClaw ha corregido una vulnerabilidad de severidad alta en su sistema core que permitía a sitios web maliciosos conectarse a agentes de IA ejecutándose localmente e tomar control sobre ellos. La vulnerabilidad reside en la puerta de enlace OpenClaw base, sin depender de plugins, marketplace o extensiones instaladas por usuarios. Según Oasis, el problema se encuentra "exactamente como está documentado."

https://thehackernews.com/2026/02/clawjacked-flaw-lets-malicious-sites.html

Exposición de Claves API de Google Cloud con Acceso a Gemini

Relevancia: Alta

Investigadores de Truffle Security han descubierto que claves API públicas de Google Cloud (identificadas con el prefijo "AIza"), típicamente designadas como identificadores de proyecto para fines de facturación, pueden ser abusadas para autenticarse en endpoints sensibles de Gemini y acceder a datos privados. Se han identificado casi 3,000 claves API incrustadas en código del lado del cliente diseñadas para proporcionar servicios relacionados con Google.

https://thehackernews.com/2026/02/thousands-of-public-google-cloud-api.html

2. AMENAZAS Y ATAQUES DESTACADOS

Extensión de Chrome QuickLens Compromete Criptografía

Relevancia: Alta

La extensión de Chrome "QuickLens - Search Screen with Google Lens" fue comprometida y distribuida con malware mediante ClickFix attacks, intentando robar criptomonedas de miles de usuarios. La extensión fue removida de la Chrome Web Store tras detectarse la actividad maliciosa.

https://www.bleepingcomputer.com/news/security/quicklens-chrome-extension-steals-crypto-shows-clickfix-attack/

Botnet Kimwolf: Investigación del Operador "Dort"

Relevancia: Alta

En enero de 2026, se reveló una vulnerabilidad utilizada para ensamblar Kimwolf, identificado como el botnet más grande y disruptivo del mundo. El operador bajo el alias "Dort" ha coordinado ataques masivos de negación de servicio distribuido (DDoS), doxing, inundación de correos electrónico contra investigadores de seguridad y periodistas, llegando incluso a coordinar llamadas SWAT a residencias. El análisis examina información pública disponible sobre la identidad del operador.

https://krebsonsecurity.com/2026/02/who-is-the-kimwolf-botmaster-dort/

Robo de $4.8M en Criptomoneda por Exposición de Frase Semilla

Relevancia: Alta

El Servicio de Impuestos Nacional de Corea del Sur expuso accidentalmente la frase mnemotécnica de recuperación de una billetera de criptomoneda incautada en un comunicado oficial, permitiendo a atacantes robar 6.4 mil millones de won (USD $4.8M) en criptomonedas.

https://www.bleepingcomputer.com/news/security/48m-in-crypto-stolen-after-korean-tax-agency-exposes-wallet-seed/
